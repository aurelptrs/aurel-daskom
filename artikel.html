Pengalaman Praktik Mata Kuliah Dasar Komputasi



Saya merupakan mahasiswa baru Teknik Informatika Udinus. Mata kuliah Dasar Komputasi berlangsung pada setiap hari

kamis di gedung D. Setiap minggu nya, para mahasiswa mendapat arahan dari dosen untuk mengerjakan praktikum.

Selain itu, beliau juga memberikan beberapa materi pembelajaran yang asing bagi saya. Contoh nya seperti materi Coding.


Materi favorit saya pada mata kuliah ini adalah saat mengerjakan materi excel. Sebelumnya saya sudah pernah berkutat

dengan aplikasi excel. karena pekerjaan saya yang memang berhubungan. Sedari SMA, saya gemar merekap sesuatu menggunakan

excel dan spreadsheets. Begitu mengerjakan praktik di kampus, saya tidak asing lagi dengan software tersebut. Saya ingin

mempelajarinya nya lebih dalam lagi.


Suasana yang kondusif serta ruangan yang tentunya nyaman membuat saya senang dalam mengikuti mata kuliah ini. Tidak perlu

takut jika tidak dapat memahami materi, karena dosen dan para asisten lab siap membantu mahasiswa selama pembelajaran berlangsung.


Daftar Materi Praktik Paling Favorit

1. Fungsi-fungsi Excel

2. Menampilkan Grafik

3. Penulisan Ilmiah

4. Merakit PC Secara Virtual

5. Fun Code with Coder

6. Penulisan HTML
